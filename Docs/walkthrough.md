# 채굴기 제어 UI 사용 가이드

## 개요
이 프로그램은 XMRig와 Rigel 채굴기를 윈도우 데스크톱 환경에서 손쉽게 제어하고 모니터링하기 위해 제작되었습니다.

## 설치 및 실행 (배포 버전)
1.  **압축 해제**: 제공된 `Dist` 폴더를 원하는 위치에 복사하거나 압축을 풉니다.
2.  **실행**: 폴더 내의 `EncryptionMinerControl.exe`를 실행합니다.

## 화면 구성
### 1. Dashboard (현황판)
프로그램의 현재 상태를 한눈에 볼 수 있습니다.
*   **상태 카드**: XMRig(CPU)와 Rigel(GPU)이 현재 켜져 있는지 꺼져 있는지 큰 아이콘으로 보여줍니다.
*   **Idle Monitor**: 마우스를 움직이지 않을 때 시간이 얼마나 흘렀는지 **게이지(Loading Bar)**로 보여줍니다. Bar가 꽉 차면(60초) 자동으로 채굴이 시작됩니다.
*   **Keep Awake Monitor**: 절전 방지(Jiggler) 기능이 켜져 있을 때, 다음 마우스 움직임(지글링)까지 남은 시간을 **게이지**로 보여줍니다. (움직이면 게이지가 꽉 차고 다시 리셋됩니다.)

### 2. Monitoring (제어 및 로그)
실제 채굴기를 조작하고 로그를 확인하는 곳입니다.
*   **로그 창**: 실시간 채굴 로그가 표시됩니다.
*   **START/STOP**: 채굴기를 수동으로 켜거나 끕니다.

### 3. Settings (설정)
*   **Auto Detect**: 채굴기 경로 및 지갑 설정을 자동으로 잡습니다.
*   **Reset to Batch**: 기존 배치 파일 설정으로 되돌립니다.
*   **General Settings**:
    *   **Keep Awake (Mouse Jiggler)**: 절전 모드 진입을 방지하는 기능을 켜고 끕니다. 주기는 기본 60초이며 최소 5초까지 설정 가능합니다.

## 초기 설정 (Auto Config)
1.  **[Settings]** 탭 → **"AUTO DETECT & CONFIGURE"** 클릭.
2.  **"SAVE SETTINGS"** 클릭. 끝!
